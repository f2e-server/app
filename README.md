# app
create-f2e-server-app model via webpack + ts + react

## usage
1. webpack compile
2. less auto enabled
3. livereload
4. simple server
   1. with route
   2. with output types
5. agent see [.f2econfig.js](.f2econfig.js)

## install
npm install

## dev
npm start

## prod
npm start build

## output frontend resources
npm run build
